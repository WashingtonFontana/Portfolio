# Washington Fontana Netto

Tenho 28 anos e resido em Juiz de Fora-MG. Atualmente, estou cursando o curso Superior de Tecnologia em Análise e Desenvolvimento de Sistemas pela Uninter e atuo como desenvolvedor Front-End freelancer, criando interfaces funcionais e modernas enquanto aprimoro continuamente minhas competências técnicas.

## Objetivo

Apresentar o meu trabalho e posicionamento profissional de forma clara para recrutadores, clientes e outros desenvolvedores, destacando:

- atuação atual como desenvolvedor Front-End freelancer;
- projetos publicados com links de demonstração;
- trajetória de estudos em desenvolvimento web;
- contato direto via WhatsApp;
- currículo em PDF disponível no projeto.

## Funcionalidades existentes

- Navegação interna por âncoras para as seções principais.
- Hero com chamada principal, botões de ação e resumo profissional.
- Seção de apresentação com proposta de valor e diferenciais.
- Lista visual de tecnologias exibidas na interface.
- Seção “Sobre” com informações profissionais e educacionais.
- Cards de projetos com imagem, descrição, stack apresentada e link de demonstração.
- Seção de credenciais/formações.
- Formulário de contato com validação no navegador.
- Geração de mensagem formatada para WhatsApp via JavaScript.
- Botão flutuante de contato por WhatsApp.
- Uso de imagens otimizadas em WebP com fallback para PNG/JPG.
- Recursos básicos de SEO, incluindo meta tags, `robots.txt` e `sitemap.xml`.
